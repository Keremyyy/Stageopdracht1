## Grid maken

- Maak values aan voor elke kleur (nu nog geen kleur) 
- maak een array van kleur values
- zorg ervoor dat elk grid vakje een kleur value krijgt. Het 9e vakje wordt of een vaste of een random kleur
## Gameplay
- maak een punten teller die bijhoudt hoe vaak je hebt goed gelikt (gewoon een int die steeds +1 gaat)
- pak een random kleur uit de array van kleuren
- Schrijf deze kleur met rode letters boven de grid
- maak een on click event die ckeckt of je op de jusite kleur hebt geklikt en dan +1 doet 
- maak een timer die aftelt
